# VellaVeto Brand Kit (draft)

## What this logo is trying to say
A **policy gateway / firewall** for agent tool calls: a **shield-like hexagon** with a bold **V** and a **horizontal “veto” bar** (deny-by-default).  
This aligns with VellaVeto’s positioning as a security-first control plane for MCP and agent tool calls.

## Color palette
- Navy: #0B1020
- Ink (text on light): #121826
- Magenta (primary accent): #C04BF0
- Cyan (secondary accent): #32D1FF
- Slate (neutral): #9AA4B2
- Fog (text on dark): #E6EAF2

## Typography
- UI / marketing: **Inter** (fallback: system-ui)
- Code / snippets: **JetBrains Mono** (fallback: ui-monospace)

## Usage rules (practical)
- Prefer **SVG** whenever possible.
- Maintain clear space around the mark: at least **¼ of the icon width**.
- If the logo is < 24px tall, use the **logomark only**.
- On dark backgrounds, use the `*-dark.*` variants.

Embed `docs/readme-header.png` at the top of the GitHub README for a clean header.

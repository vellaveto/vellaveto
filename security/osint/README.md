# OSINT Supply Chain Notes

Drop intelligence briefs about suspicious packages, typosquats, or malicious registry activity here. Each file should include:

- `Source:` (vendor, newsletter, report)
- `Date:` when the intel hit
- `Summary:` what changed (e.g., "typosquatted PyPI package `urllib3-security` contains exfil payload")
- `Action:` how the team responded or will respond

The dependency monitor script prints the directory contents when it runs; keeping this folder current keeps the OSINT reminders visible.

Zero-config protection

Pick a level and go — no policy files, no security domain knowledge needed.

ShieldEntry-level

Blocks credentials + dangerous commands. Injection and DLP scanning in blocking mode. Default allow.

vellaveto-proxy --protect shield -- ./server
FortressRecommended

Shield + exfil domain blocking, AI config protection (SANDWORM defense), git hook blocking, approval gates for destructive ops.

vellaveto-proxy --protect fortress -- ./server
VaultMaximum

Fortress with default deny. Must whitelist what you need. Shadow agent detection enabled.

vellaveto-proxy --protect vault -- ./server

What It Does

Complete mediation between AI agents and their tools across every transport.

Govern

Centralize policy, ABAC rules, approval workflows, and tenant controls. Cedar-compatible policy import/export.

Enforce

Inline evaluation on stdio, HTTP, WebSocket, gRPC, and SSE. Injection detection, DLP scanning, Wasm plugins.

Discover

Live topology graph of MCP servers and tools. Fail-closed on unknown tools. Drift detection and re-crawl scheduling.

Prove

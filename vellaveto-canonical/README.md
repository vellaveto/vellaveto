# vellaveto-canonical

Canonical security policy presets for the [Vellaveto](https://vellaveto.online) MCP security gateway.

## Overview

Provides ready-to-use policy presets for common scenarios:

- File system access controls
- Network domain allowlists/blocklists
- Tool invocation rate limiting
- SANDWORM hardening preset (10 defensive layers)

## Usage

```toml
[dependencies]
vellaveto-canonical = "6"
```
